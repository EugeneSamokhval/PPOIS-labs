1.Вводим путь к файлу сохранения
2. Вводим одну из комманд
2.1. next_turn запускает симуляцию
2.2. show Отобржает игровое поле
2.3. save сохраняет состояние поля в указанный файл
2.4. load затружает состояние поля из файла
2.5. add_creature создаёт выбранное существо
2.6. exit выходит из симуляции
